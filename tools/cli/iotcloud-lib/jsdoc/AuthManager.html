AuthManager

Authentification and Authorization Manager. Call init() before use.

new AuthManager(configurator)

Create an AuthManager.
Parameters:
Name Type Description
configurator Configurator A Configurator to manipulate configuration.

Methods

init( [iotCloudAgent] [, authProviderAgent] [, iotConsoleAgent])

Set private field from configuration. And create agent for cloud
Parameters:
Name Type Argument Description
iotCloudAgent IotCloudAgent <optional>
 IotCloudAgent to use.
authProviderAgent AuthProviderAgent <optional>
 AuthProviderAgent to use.
iotConsoleAgent IotConsoleAgent <optional>
 AuthProviderAgent to use. If not specified, create default agent from configuration.

refreshToken(refreshOnlyWhenExpired)

This method loads data for refresh access token from configurator. Then it sends a request to get access token to the IoT Cloud through the agent.

Name Type Default Description
