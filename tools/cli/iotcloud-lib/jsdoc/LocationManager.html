name string A nickname given for the location (eg. Home)

latitude string | number <optional>
 A geographical latitude. (i.e 10.0, '10.0')
longitude string | number <optional>
 A geographical longitude. (i.e 10.0, '10.0')

 The radius in meters around latitude and longitude which defines this location.
